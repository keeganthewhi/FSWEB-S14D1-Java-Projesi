## About

Covers the inheritance mechanism in Java, demonstrating how classes can extend other classes to reuse and specialize behavior. This assignment builds hierarchies of related types using `extends`.

## Key Concepts

- Class inheritance with `extends`
- Method overriding and `@Override`
- The `super` keyword
- Constructor chaining in inheritance hierarchies
